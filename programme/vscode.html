# vscode 快捷键

  1. 查找文件
  • Mac: CMD + P
  • Windows/Linux: CTRL + P
  1. 将光标添加到所有匹配的选择
  • Mac:CMD + SHIFT + L
  • Windows/Linux: CTRL + SHIFT +L
  1. 选中当前行
  • Mac: CMD + L
  • Windows/Linux: CTRL + L
  1. 转到特定行
  • Mac: CTRL + G
  • Windows/Linux: CTRL + G
  1. 关闭所有打开的编辑器选项卡
  • Mac: CMD + K + W
  • Windows/Linux: CTRL + K + W
  1. 切换块评论
  • Mac: SHIFT + OPTION + A
  • Windows: SHIFT + ALT + A, Linux: CTRL+SHIFT+A
  1. 折叠代码块
  • Mac: CMD + OPTION + [
  • Windows/Linux: CTRL + SHIFT + [
  1. 打开集成终端
  • Mac: CTRL + `
  • Windows/Linux: CTRL+ `
  1. 拆分编辑器视图
  • Mac: CMD + \
  • Windows/Linux: CTRL+\
  1. 切换侧边栏
  • Mac: CMD + B
  • Windows/Linux: CTRL+B
  1. 向上复制
  2. 向下复制
  3. 删除行
  4. 选中行头、行尾
  5. 光标移动
  6. 选择单词
